### Setup
1. 将 Resources/Templates，Projects 两个文件夹copy到你的Obsidian Vault
2.  .obsidian/snippets 文件夹内容，copy到你的.obsidian/snippets
 这里主要用到MCL Multi Column.css 这个文件，要在设置-外观-snppets 这里启用它
<img width="2298" height="1430" alt="image" src="https://github.com/user-attachments/assets/c7762579-4543-4a83-b461-385baee02a32" />
 
4.  .obsidian/themes 文件夹内容,copy到你的  .obsidian/themes
5. 配置DailyNote插件：指定模版为Resources/Templates/DearDiary.md
6. 配置主题：设置为DaySpring
7. 如果要改日记模版里打卡的项目，记得把打卡项目的tag一起改掉。比如 study #habit/study 改成 read #habit/read

应该就ok啦！！
