### Setup
1. 将 Resources/Templates，Projects 两个文件夹copy到你的Obsidian Vault
2.  .obsidian/snippets 文件夹内容，copy到你的.obsidian/snippets
3.  .obsidian/themes 文件夹内容,copy到你的  .obsidian/themes
4. 配置DailyNote插件：指定模版为Resources/Templates/DearDiary.md
5. 配置主题：设置为DaySpring

应该就ok啦！！
